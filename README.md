# 모션인식을 활용한 G러닝 MGMG

모션인식을 활용한 G러닝 <MGMG>는 웹캠 기반의 모션인식 게임을 통해 학습하는 G러닝 방식을 지원합니다. <br/>
이를 통해 피교육자의 학습 동기와 몰입도를 향상시킬 수 있습니다.

## 개발기간
2022년 03월 ~ 11월

## 적용 기술
Phaser3, JavaScript, MediaPipe
